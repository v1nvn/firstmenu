# Privacy Policy

**Last Updated:** January 6, 2026

## Overview

**firstmenu** ("the App") is a menu bar system monitoring application for macOS. We are committed to protecting your privacy and being transparent about the data we collect and how we use it.

## Information We Collect

### No Personal Data Collection

**firstmenu does NOT collect, store, or transmit any personal information about you, including:**

- No user accounts or authentication
- No personal identification information (name, email, address)
- No usage analytics or tracking
- No crash reports or diagnostics sent to external servers

### System Information (On-Device Only)

The App reads the following system information for display purposes only:

- **CPU Usage:** Current processor utilization percentage
- **Memory Usage:** RAM used vs. total available
- **Storage Usage:** Disk space used vs. total available
- **Network Activity:** Current upload/download speeds
- **Running Applications:** List of active applications
- **System Sleep State:** Whether sleep prevention is active

This information is:
- Displayed only in the App's interface
- Stored temporarily in memory while the App is running
- Never written to disk (except for standard macOS caching)
- Never transmitted to any external server

### Weather Data

The App fetches weather information from:

- **Source:** Open-Meteo API (open-meteo.com)
- **Location Determination:** IP-based geolocation (automatic)
- **Data Collected:** Your approximate location based on IP address

**How it works:**
1. Your IP address is sent to the Open-Meteo API to determine your approximate location
2. Weather data for your location is returned and displayed
3. No personally identifiable information is stored or shared

**Open-Meteo's Privacy Policy:** https://open-meteo.com/en/privacy

## Data Storage

### Local Storage

The App stores minimal data locally:

- **User Preferences:** Weather refresh interval, menu bar visibility settings
  - Stored in: macOS UserDefaults (standard system location)
  - Format: Key-value pairs
  - Contains: No personal information

### No Cloud Storage

- We do not use any cloud services for data storage
- No data is synced to any external servers
- No backups of your data are maintained by us

## Data Sharing

**We do NOT sell, rent, or share your information with third parties for marketing purposes.**

The only external service the App uses is:

- **Open-Meteo API:** For weather data (see above)

## Data Security

While we take reasonable precautions to protect your information:

- All data remains on your device
- Standard macOS app sandboxing provides isolation
- No network encryption is needed (no data is transmitted except weather API calls)

## Third-Party Services

The App integrates with the following third-party service:

### Open-Meteo (Weather Data)

- **Purpose:** Display current weather conditions
- **Data Sent:** IP address (for location approximation)
- **Privacy Policy:** https://open-meteo.com/en/privacy
- **Your Choice:** Cannot be disabled (core feature)

## Children's Privacy

The App is not directed to children under 13. We do not knowingly collect information from children under 13.
